#beimi
开源棋牌游戏，包含麻将、德州、斗地主
首个版本会采用当前最流行的房卡模式

贝密游戏是一系列棋牌游戏的名称，其中包含麻将、斗地主、德州，目前正在进行UI设计以及后台系统（JAVA）开发，月份发布0.0.1版本，仅包含前端UI方案，预计8月份发布首个版本，敬请关注！


1. 开发工具：Cocos Creater
1. 开发语言：Java + JavaScript
1. 服务端框架：Spring Boot + MySQL + JPA + NettySocketIO
1. 客户端语言：Cocos Creater/JavaScript
